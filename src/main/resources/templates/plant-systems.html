افزودن سیستم واحد
برای زیرسیستم‌ها (مثلاً HVAC زیر برق) سیستم والد را انتخاب کنید.
فقط برای سیستم ریشه لازم است؛ زیرسیستم مکان را از والد به‌ارث می‌گیرد.

ورود از اکسل

ستون‌های مورد نیاز (ردیف اول سرستون):

code | name | parentSystemCode | locationCode

برای سیستم ریشه locationCode کافی است. برای زیرسیستم parentSystemCode را پر کنید (مکان از والد به‌ارث می‌رسد). والد باید در ردیف‌های بالاتر فایل تعریف شده باشد.
