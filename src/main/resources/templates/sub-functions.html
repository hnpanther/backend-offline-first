افزودن تابع فرعی
هر تابع فرعی فقط یک والد مستقیم دارد؛ زنجیرهٔ بالادستی خودکار پر می‌شود.
ویرایش تابع فرعی
تغییر والد مستقیم، زنجیرهٔ بالادستی را به‌روز می‌کند.
ورود از اکسل

ستون‌های مورد نیاز (ردیف اول سرستون):

code | name | tag | parentSubFunctionCode | mainFunctionCode | systemCode | locationCode

اولویت والد: parentSubFunctionCode سپس mainFunctionCode سپس systemCode سپس locationCode.

  • اگر parentSubFunctionCode پر باشد → زیر تابع فرعی دیگر
  • اگر mainFunctionCode/Name پر باشد → زیر تابع اصلی
  • اگر systemCode/Name پر باشد → مستقیم زیر سیستم واحد
  • اگر locationCode/Name پر باشد → مستقیم زیر مکان
