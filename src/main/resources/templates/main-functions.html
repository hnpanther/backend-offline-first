افزودن تابع اصلی
تابع اصلی زیر یک سیستم، یک مکان، یا یک تابع اصلی دیگر قرار می‌گیرد؛ زنجیرهٔ بالادستی خودکار پر می‌شود.
ویرایش تابع اصلی
تغییر والد مستقیم، زنجیرهٔ بالادستی را به‌روز می‌کند.
ورود از اکسل

ستون‌های مورد نیاز (ردیف اول سرستون):

code | name | parentMainFunctionCode | systemCode | locationCode

  • اگر parentMainFunctionCode پر باشد → زیر تابع اصلی (system/location از والد)
  • در غیر این صورت اگر systemCode پر باشد → زیر سیستم واحد
  • در غیر این صورت اگر locationCode پر باشد → مستقیم زیر مکان
